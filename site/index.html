Bem-vindo à sua Página de Teste Pessoal

Esta página é o ponto de partida (index) do seu projeto.

Links de Exemplo

Use estes links para navegar e testar diferentes seções do seu site. Você precisará criar os arquivos correspondentes.

  • Sobre Mim - Informações pessoais e de contato.
  • Portfólio / Projetos - Exposição dos seus trabalhos.
  • Contato - Formulário ou detalhes de contato.
  • Blog (Pasta) - Link para uma pasta (diretório) de posts.
  • Link Externo (Google) - Exemplo de link para outro site.

Estrutura do Projeto

Para que os links internos funcionem, sua estrutura de pastas deve ser assim:

  • `index.html` (Este arquivo)
  • `sobre.html`
  • `portfolio.html`
  • `contato.html`
  • `blog/` (Pasta que pode conter outro `index.html`)
